# AI Health Symptom Checker

## Introduction

AI Health Symptom Checker is a Python-based project that predicts possible health conditions based on the symptoms entered by the user. It uses a machine learning model trained on a health dataset to provide a possible prediction and basic health suggestions.

## Features

- Takes symptoms as input
- Predicts possible disease
- Generates a health report
- Displays simple health suggestions
- Creates basic data visualizations

## Technologies Used

- Python
- Pandas
- Scikit-learn
- Matplotlib

## Project Structure

AI_Health_Symptom_Checker/

- app.py
- symptom_checker.py
- dataset.csv
- report_generator.py
- visualizations.py
- requirements.txt
- README.md

## How to Run

1. Install the required libraries.
2. Run `app.py`.
3. Enter symptom values as instructed.
4. The program predicts the possible disease and generates a health report.
